# 🎨 Daily CSS Challenge

This repository contains my **Daily CSS Challenges** where I practice and improve my CSS skills by building small UI components, layouts, and animations.

The goal of this challenge is to **become better at CSS through consistent daily practice**.

## 🚀 About the Challenge

Every challenge focuses on creating a small UI element using **HTML, CSS, and sometimes JavaScript**.
These projects help in understanding:

* CSS Layouts (Flexbox, Grid)
* Responsive Design
* Animations and Transitions
* UI Components
* Modern CSS techniques

## 💡 Who Can Try This?

Anyone who wants to **improve their CSS skills** can try these challenges.
Feel free to clone the repository and practice along.

## 📂 Project Structure

Each challenge will have its own folder.

Example:

```
daily-css-challenge
│
├── challenge-01
│   ├── index.html
│   ├── style.css
│
├── challenge-02
│   ├── index.html
│   ├── style.css
│
└── challenge-03
    ├── index.html
    ├── style.css
```

## 🛠 Technologies Used
